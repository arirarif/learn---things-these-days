# 09 — Finance Concept Glossary (বাংলা)

Stage 0-এর ১০টা foundational concept। প্রতিটা read → watch → write → teach → tick পদ্ধতিতে শিখুন।

---

## ১. Cash flow — নগদ প্রবাহ

মাসিক income − ব্যয়। Positive = surplus। Negative = ground হারাচ্ছেন।

**উদাহরণ:** আয় ৭০k, খরচ ৬৫k → cash flow +৫k। বছরে ৬০k surplus — এটাই savings/investing/debt clearing-এর উৎস।

**Test:** *গত মাসে আপনার cash flow ঠিক কত ছিল?*

---

## ২. Net worth — মোট সম্পদ

= যা আছে (asset) − যা ঋণ (liabilities)। Income মাসিক, wealth সঞ্চিত।

**উদাহরণ:** ৫,০০০ টাকা cash + ০ investment − ৭,০০,০০০ debt = −৬,৯৫,০০০। Uncomfortable, কিন্তু এটাই starting line।

**Test:** *এখন আপনার net worth কত — বলতে পারেন?*

---

## ৩. Compound interest — যৌগিক সুদ

সুদের উপর সুদ। দু'দিকেই কাজ করে।

**Einstein-কে quote করে বলা হয়:** *"Compound interest দুনিয়ার অষ্টম আশ্চর্য। যে বোঝে সে earn করে। যে বোঝে না সে pay করে।"*

**উদাহরণ:** ৩০ বছরে ১ লাখ @ ১০% = ১৭.৪৫ লাখ। উল্টোদিকে: ৫ বছরে ১ লাখ @ ৬০% (unpaid) = ~১০.৫ লাখ। উচ্চ-rate debt তাই এত dangerous।

**Test:** *৬০% APR debt কেন ৬% APR debt-এর চেয়ে ১০x না, exponentially বেশি খারাপ?*

---

## ৪. Inflation — মূল্যস্ফীতি

টাকা সময়ের সাথে purchasing power হারায়।

**Bangladesh এখন:** ~৮–১০%। আজকের ১০০ টাকা পরের বছরের ~৯১ টাকার সমান।

**উদাহরণ:** ১,০০,০০০ টাকা ৫ বছর mattress-এর নিচে → nominal-এ ১,০০,০০০ থাকে, কিন্তু purchasing power ~৬৫,০০০-তে নেমে আসে। ৩৫% value silently হারিয়ে যায়।

**Test:** *আপনার savings account inflation-কে beat করে?*

---

## ৫. Real return — প্রকৃত return

= Nominal return − inflation।

৫% savings, ৯% inflation → real return **−৪%**।

**Bangladesh Sanchaypatra** ~১১%, inflation ৯% → real return ~২%। বাংলাদেশে এটা "ভালো" কিন্তু আসলে wealth growth শুধু ২%।

**Test:** *যেকোনো investment-এর nominal return শুনে real return calculate করতে পারেন?*

---

## ৬. Time value of money — সময়ের মূল্য

আজকের ১ টাকা > পরের বছরের ১ টাকা। কারণ আজকের টাকা invest করা যায়, এবং inflation পরের বছরের টাকার value কমায়।

**উদাহরণ:** কেউ বলছে "৩ বছরে ৫০k দেব আজকের ৩০k-এর বদলে।" ৩০k-কে ১০%-এ ৩ বছর invest করলে → ৩৯,৯৩০। মানে ৫০k > ৩০k by a lot। *এভাবেই loan/insurance/investment evaluate করা হয়।*

**Test:** *নিজের ভাষায় বলতে পারেন কেন টাকার time value আছে?*

---

## ৭. Asset vs Liability — সম্পদ vs দায়

**Robert Kiyosaki:**
- Asset → pocket-এ টাকা আনে
- Liability → pocket থেকে টাকা নেয়

**উদাহরণ:** আপনার freelance laptop = asset (income আনে)। Credit card balance = liability। আপনার skill = সবচেয়ে বড় asset।

**Test:** *আপনার ৩টা actual asset (যা টাকা আনে) আর ৩টা actual liability নাম বলুন।*

---

## ৮. Income vs Wealth — আয় vs সম্পদ

- Income = মাসিক আসা টাকা
- Wealth = সময়ের সাথে accumulated

**গল্প:** ঢাকার ৫ লাখ income / ৪.৮ লাখ ব্যয়ের ডাক্তার < ৫০k income / ৪০k ব্যয়ের শিক্ষক যিনি ২০ বছর invest করেন। Income flatters today. Wealth defines tomorrow।

**Test:** *আপনি income-কে wealth-এ convert করছেন কি — কোনো rate-এ?*

---

## ৯. Risk vs Return — ঝুঁকি ও মুনাফা

**নিয়ম:** উচ্চ return = উচ্চ risk। কোনো ব্যতিক্রম নেই।

- Low: savings, sanchaypatra, government bond (~৫–১১%)
- Medium: diversified stock, real estate (~১২–১৫% long-term)
- High: single stock, crypto, forex (+৫০% বা −৯০%)

**মিথ্যা চিনুন:** "Guaranteed ৩০% return" — exist করে না। Either risky (lying about risk), leveraged (won't last), or fraud।

**Test:** *"Guaranteed" investment-এ সর্বোচ্চ কত return বিশ্বাস করা উচিত? (উত্তর: Sanchaypatra rate-এর মতো। এর বেশি = scam।)*

---

## ১০. Halal vs Haram Finance

**Haram:**
- *Riba* (সুদ) — কোনো loan-এ সুদ আদান-প্রদান
- *Gharar* (অতিরিক্ত অনিশ্চয়তা) — gambling, forex speculation
- Haram industry-র income

**Halal alternative:**
- *Mudaraba* — profit-sharing partnership
- *Musharaka* — joint partnership
- *Sukuk* — Islamic bond
- *Wadiah / Mudaraba savings* — Shariah-compliant savings

**Bangladesh halal bank:** IBBL, Al-Arafah, EXIM, City Islami, First Security Islami, Social Islami, Standard Bank Islamic।

**Test:** *Conventional interest-based savings account-এর ৩টা halal alternative নাম বলতে পারেন?*
